This is my Personal Project.

Project Name – Twitter Clone


The Technologies with which I worked with while making personal project ->
  1. Python
  2. Flask
  3. HTML
  4. CSS
  5. Wtf-forms
  6. Sql-Alchemy
  7. Bootstrap
  8. Jinja
Apart from these, I have gone through flask-modules, javascript and jquery stuff over, through online websites.

The functionalities that my project cover are –
  1. Login/Logout/Register
    2. Along with Login/Logout, I have also used various validators(database validations and frontend validations using boostrap) for validating the login and logout form. Also, I have encrypted the password which will be done at the time of registration. All validations will be done at the time of registration also.
  2. CRUD (create, update, delete)
    3. We can update the post, delete the post, and create a new post. Also, I have added a feature of having profile picture in the post(note – name of picture will be saved using random hex numbers generated everytime with the help of python module). Also, I have added few conditions such as one cant do CRUD operations until he is logged in, and if he triggered any of these events before login, he will be redirected to login/register page.
  3. Custom Error Pages
    4. I have created custom error pages, which will be displayed if a wrong url or page is triggered before fulfilling the condition.
  4. Author Specific Posts
    5. Added a functionality, such that on clicking on the name of the author, we can view all the posts posted by that particular author.
  5. SideBar filters
    6. Added a functionality of filter the post by title search, date search and author search.
  6. Pagination functionality
    7. Added Pagination functionality – in this we can see only a specific number of pages at a time and rest all the pages, can be shown by moving to those page numbers.
  7. Database
    8. Everything functionality is connected with the database(sql-alchemy).
  8. Python & Flask modules
    9. Learned about Packages, routing, models, modules, blueprint and configurations, and tried to follow the MVC structure. And also learned about various other flask and flask related modules and python modules.
  9. Forgot & Reset Password
    10. Added a functionality of forget password and reset password and reset email using python module.
  10. ...
